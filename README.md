
# NLM Bot

Project built on DiscordJS v13.

The bot retrieves and displays Rainbow Six Siege stats in your
server, thanks to [**r6api.js**](https://www.npmjs.com/package/r6api.js) package and Ubisoft's public API.

There are a variety of commands, you can use **$help** to see all of them.

# Installation
```
npm install
```

# Launch
```
node index.js
```
or

```
nodemon index.js
```
___

# Rainbow Six Tracker
### Commands
Seasonal RANKED Summary:

    $r6 [nickname]

Seasonal RANKED stats (more informative):
    
    $ranked [nickname]

Seasonal CASUAL stats:
    
    $casual [nickname]

OVERALL R6S stats:
    
    $general [nickname]


### Examples

    $r6 Shaiiko.BDS
    $general ThinkinNade.SSG
    $casual SHA77ELELE
